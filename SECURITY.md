# Security Policy

## Reporting a Vulnerability
- Email: security@typosentinel.example
- Preferred format: detailed description, reproduction steps, impact assessment
- We will acknowledge receipt within 48 hours and provide status updates until resolution.

## Supported Versions
- We support the latest minor release and security patches.

## Disclosure Policy
- Coordinated disclosure with reporters.
- We aim to provide fixes within a reasonable timeframe based on severity.

## Known Limitations
- Some API endpoints are marked as not implemented and return `501`.
- Demo mode endpoints are flagged with `X-Demo-Mode: true`.
