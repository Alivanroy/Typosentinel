Comprehensive Security Features

Typosquatting Detection

Advanced ML algorithms detect packages with names similar to popular libraries, protecting against typosquatting attacks.

Malware Detection

Real-time scanning for malicious code patterns, suspicious behaviors, and known threat signatures.

Multi-Ecosystem Support

Supports npm, PyPI, Go modules, Maven, NuGet, and more package ecosystems with unified scanning.

Reputation Analysis

Evaluates package reputation based on maintainer history, download patterns, and community feedback.

Real-time Scanning

Lightning-fast scanning with intelligent caching and optimized algorithms for CI/CD integration.

Developer-Friendly API

RESTful API with comprehensive documentation for seamless integration into your development workflow.

API Documentation

Scan Package

POST /api/v1/scan 
{
  "ecosystem": "npm",
  "package": "package-name",
  "version": "1.0.0"
}

Batch Scan

POST /api/v1/scan/batch 
{
  "packages": [
    {"ecosystem": "npm", "package": "lodash"},
    {"ecosystem": "pypi", "package": "requests"}
  ]
}

Get Scan Results

GET /api/v1/scan/{scanId} 
{
  "scanId": "uuid",
  "status": "completed",
  "riskScore": 0.85,
  "threats": [...]
}

API Features

  • RESTful API with JSON responses
  • Rate limiting and authentication
  • Webhook support for async scanning
  • Comprehensive error handling
  • OpenAPI 3.0 specification
